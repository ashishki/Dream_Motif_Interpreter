# Гайд пользователя — Dream Motif Interpreter

Актуально на 2026-04-21

## Что это вообще такое

Это личный помощник для работы со снами.

Он умеет:

- хранить архив снов
- искать по нему
- отвечать на вопросы о повторяющихся образах и темах
- принимать голосовые сообщения в Telegram
- сохранять новый сон из чата
- автоматически подтягивать изменения из Google Docs

Главное:

- бот не ставит диагнозы
- бот не объясняет, что сон "на самом деле значит"
- бот помогает навести порядок в архиве и увидеть связи

## Где хранятся сны

Сейчас есть два источника:

1. Google Docs
2. Telegram

### Google Docs

Google Docs остаётся основным дневником.

Если ты добавил или изменил сон в документе, система сама проверяет, изменился ли файл, и только после этого подтягивает обновления в базу.

Это не мгновенно, но обычно занимает несколько минут.

### Telegram

Через Telegram можно:

- задавать вопросы про архив
- отправлять голосовые сообщения
- сохранить новый сон как новую запись

Через Telegram сейчас нельзя:

- редактировать уже существующий сон
- удалять старый сон
- менять архив "по ходу разговора" без явной просьбы сохранить новый сон

Это сделано специально, чтобы архив не ломался от случайных действий.

## Как пользоваться ботом

Писать можно обычным человеческим языком.

Примеры:

- "Что мне снилось на прошлой неделе?"
- "Найди сны, где была вода"
- "Какие темы повторялись чаще всего?"
- "Покажи мои последние сны"
- "Сохрани новый сон"

Никакие теги, команды и специальный синтаксис не нужны.

## Как сохранить новый сон через Telegram

Если хочешь добавить новый сон, нужно сказать об этом прямо.

Подходят фразы вроде:

- "Сохрани новый сон"
- "Запиши сон"
- "Добавь новый сон в архив"

После этого отправь текст сна.

Что произойдёт дальше:

1. Сон сохранится как новая запись.
2. Система автоматически обработает его.
3. Он появится в архиве и станет доступен для поиска.

Важно:

- бот создаёт только новую запись
- если такой текст уже был сохранён раньше, система не станет плодить дубликаты
- бот не редактирует старые записи

## Как работают изменения в Google Docs

Система не перечитывает весь документ без повода.

Она делает короткую проверку:

1. смотрит, изменился ли файл в Google Docs
2. если файл не менялся, ничего не делает
3. если файл изменился, запускает обычную синхронизацию

Это значит:

- лишней нагрузки почти нет
- база обновляется только когда документ реально менялся
- если ты добавил новый сон в Google Docs, он сам появится в системе после следующей проверки

## Что можно спрашивать у бота

Бот хорошо подходит для таких задач:

- найти сон по образу, теме или настроению
- вспомнить, что снилось недавно
- посмотреть повторяющиеся темы
- сравнить похожие сны
- спросить про мотивы и параллели

Примеры:

- "В каких снах я куда-то опаздывал?"
- "Были ли у меня сны про лестницы?"
- "Что чаще повторяется: вода или дорога?"
- "Есть ли похожие сны про закрытые двери?"

Если в архиве нет ответа, бот так и скажет.

## Голосовые сообщения

Можно отправлять голосовые сообщения.

Как это работает:

1. ты отправляешь голосовое
2. система переводит речь в текст
3. бот отвечает как на обычный текстовый вопрос

Голосовой файл не хранится постоянно. Он удаляется после обработки.

## Мотивы и культурные параллели

Если эти функции включены, бот может:

- показать абстрактные мотивы сна
- затем, по твоему подтверждению, поискать культурные или мифологические параллели

Важно:

- мотивы — это не "истина о сне", а вычислительные предположения
- внешний поиск запускается только после явного подтверждения
- найденные параллели — это материал для размышления, а не готовая трактовка

## Что бот не делает

Бот специально ограничен.

Он не должен:

- выдумывать факты, которых нет в архиве
- менять старые записи "по смыслу"
- удалять или переписывать сны
- делать вид, что знает твою внутреннюю жизнь лучше тебя

Если ответ не найден, бот должен сказать это прямо.

## Оценка ответов

После содержательного ответа бот может предложить оценить его.

Можно ответить:

- `5`
- `4 Хорошо`
- `2 Слишком расплывчато`

Это нужно, чтобы постепенно подстроить стиль ответов под тебя.

## Самое важное коротко

- Google Docs — основной дневник
- Telegram — удобный интерфейс для общения с архивом
- новый сон можно сохранить через бота
- старые сны через бота не редактируются
- изменения в Google Docs подтягиваются автоматически после короткой проверки файла
- никаких команд и тегов запоминать не нужно
